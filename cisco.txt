## **შემოწმების სია:**
```
✅ 1. Server3 → 200.200.200.1 (თვითონ)
✅ 2. Server4 → 200.200.200.1 (იგივე VLAN)
✅ 3. PC VLAN 20 → 200.200.200.1 (სხვა VLAN)
✅ 4. R1 → 200.200.200.1 (დისტანციური)
✅ 5. VLAN 30 → 200.200.200.1 (III ქსელი)
```

**ერთი ხაზი:** **პინგი 200.200.200.1** ყველა ქსელიდან და **დარწმუნდი რომ მუშაობს**.



დავალება N2 ( 5 ქ)
IPv6 მინიმალური კოდი - მხოლოდ ქულების მოსაპოვებლად
1. ყველა როუტერზე (R1, R2, R3, R4):
bash
enable
conf t

! 1. სახელი
hostname sergi_tvalabeishvili

! 2. Console პაროლი
line console 0
password sergi
login
exit

! 3. Enable პაროლი
enable secret Tvalabeishvili

! 4. IPv6 ჩართვა
ipv6 unicast-routing

! 5. EIGRPv6
ipv6 router eigrp 1
 eigrp router-id X.X.X.X  ! R1=1.1.1.1, R2=2.2.2.2, R3=3.3.3.3, R4=4.4.4.4
 no shutdown
exit

end
write memory
2. R1-ზე დამატებით:
bash
conf t
! Interface to R2
interface g0/0
 ipv6 address 2001:DB8:ACAD:12::1/64
 ipv6 eigrp 1
 no shutdown

! Interface to Server1  
interface g0/1
 ipv6 address 2001:DB8:ACAD:1::1/64
 ipv6 eigrp 1
 no shutdown
